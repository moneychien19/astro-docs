## 啟動 Astro 安裝精靈

建立一個新的 Astro 網頁的首選方法是透過 `create astro` 安裝精靈。

1. 在終端機的命令列中，使用你喜歡的套件管理工具運行以下命令：

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      # 使用 npm 建立一個新專案
      npm create astro@latest
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      # 使用 pnpm 建立一個新專案
      pnpm create astro@latest
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      # 使用 yarn 建立一個新專案
      yarn create astro
      ```
      </Fragment>
    </PackageManagerTabs>

2. 輸入 `y` 確認安裝 `create-astro`
3. 當跳出提示詢問 "Where would you like to create your new project?"（你想要在哪裡建立你的新專案？）時，輸入資料夾的名稱來為你的專案建立一個新目錄，例如：`./tutorial`。

    :::note
    一個新的 Astro 專案只能被建立在一個空的資料夾中，所以請為你的資料夾選擇一個尚不存在的名稱！
    :::

4. 你會看到一個簡短的入門模板列表可供選擇，使用方向鍵（上和下）導航到 "Empty" 模板並按 Enter 鍵選擇。

5. 當跳出提示詢問 "Would you like to install dependencies?"（你想現在安裝依賴嗎？）時，輸入 `y` 確認。

6. 當跳出提示詢問你是否要使用 TypeScript，輸入 `n` 拒絕。

7. 當跳出提示詢問 "Would you like to initialize a new git repository?"（你想要現在初始化一個新的 Git 儲存庫嗎？）時，輸入 `y` 確認。

安裝精靈完成後，你就不再需要此終端機。你可以接著打開 VS Code。

## 在 VS Code 中打開你的專案

1. 打開 VS Code，系統將提示你打開一個資料夾，請選擇你在使用安裝精靈時建立的資料夾。

2. 如果這是你第一次打開 Astro 專案，你會看到一個通知詢問你是否想要安裝推薦的擴充套件，點擊通知查看它們並選擇 [Astro 語言支援擴充套件](https://marketplace.visualstudio.com/items?itemName=astro-build.astro-vscode)，它會為你的 Astro 程式碼提供語法突顯和自動補全功能。

3. 確保終端機正常運作且有出現以下提示：

    ```sh
    user@machine:~/tutorial$
    ```

    :::tip[快捷鍵]
    使用 <kbd>Ctrl + J</kbd> (macOS: <kbd>Cmd ⌘ + J</kbd>) 打開或關閉終端機。
    :::

在之後的教學中，你都可以使用 VS Code 內建的終端機，而不是你電腦上的其他終端機應用程式了。



## 在開發模式下運行 Astro

為了在開發時預覽你的網頁，你會需要在開發模式下運行 Astro。

### 啟動開發環境

1. 在 VS Code 的終端機中使用以下命令啟動 Astro 的開發環境。

    <PackageManagerTabs>
      <Fragment slot="npm">
      ```shell
      npm run dev
      ```
      </Fragment>
      <Fragment slot="pnpm">
      ```shell
      pnpm run dev
      ```
      </Fragment>
      <Fragment slot="yarn">
      ```shell
      yarn run dev
      ```
      </Fragment>
    </PackageManagerTabs>

    現在你應該會在終端機上看到 Astro 正在開發模式下運行的提示。🚀

## 查看網頁的即時預覽

你的專案檔案包含了顯示 Astro 網站所需的所有程式碼，但瀏覽器會負責將你的程式碼呈現成網頁。

1. 點擊你的終端機上出現的 `localhost` 連結以預覽你的 Astro 網頁！

    （在連接埠 `4321` 可以使用時，Astro 預設使用 `http://localhost:4321`。）

    以下是 Astro 的 "Empty" 模板在瀏覽器中呈現的樣子：

    ![A blank white page with the word Astro at the top.](/tutorial/minimal.png)

:::tip[使用 Astro 開發環境]

當 Astro 在開發模式下運行時，你將**無法**在終端機視窗中運行其他命令。相反，此視窗會在你預覽網站時給予回饋。

你可以在任何時間點使用 <kbd>Ctrl + C</kbd> 來停止運行開發環境並回到命令提示字元。

有時開發環境伺服器會在你開發時自行停止，如果此時你的即時預覽也中斷連線了，請回到終端機並輸入 `npm run dev` 重啟伺服器。
:::
