現在你已經知道你要的目標了，是時候來設置你所需的所有工具！

這個單元將向你展示如何設置你的開發環境並部署網站到 Netlify。如果你已經熟悉自己的環境和工作流程，可以直接跳到[單元二](/zh-tw/tutorial/2-pages/)。

:::tip[使用 stackblitz]

想要在線上編輯器裡完成這個教學嗎？
<details>
<summary>按照以下指示操作，然後直接進入單元二！</summary>

**設置 StackBlitz**
1. 進到 [astro.new](https://astro.new) 並點擊 "Empty Project" 下方的 "Open in StackBlitz" 按鈕以打開一個空白專案。

2. 點擊右上角的 "Sign in" 以使用你的 GitHub 帳號登入。

3. 點擊 StackBlitz 編輯器左上角的 "fork" 按鈕，將模板儲存到你的帳號中。

4. 等待專案載入完成後，你會看到 "Empty Project" 的即時預覽。

**進行修改**

在文件視窗中，你會看到 `src/pages/index.astro`，點擊將它打開後，你可以依照[寫你的第一行 Astro 程式碼](/zh-tw/tutorial/1-setup/3/) 裡的說明修改檔案。

**建立一個 GitHub 儲存庫**

1. 點擊檔案列表上方的 <kbd>Connect Repository</kbd> 按鈕並輸入你儲存庫的新名稱，然後點擊 <kbd>Create repo & push</kbd>。

2. 當你有修改需要提交到 GitHub 上，你的工作區左上方會出現一個 "Commit" 按鈕，點擊後你可以輸入提交說明並更新到儲存庫中。

**部署你的網站**

如果你想要部署到 Netlify 上，請跳到[將你的網站部署至網路上](/zh-tw/tutorial/1-setup/5/)。
否則跳到[單元二](/zh-tw/tutorial/2-pages/)開始使用 Astro！

</details>

## 我們的目標

在這個單元中，你會**建立一個儲存於 GitHub 的新專案**並**將它連接到 Netlify**。

在你編輯程式碼的過程中，你會定期提交你的修改到 GitHub 上。Netlify 會使用 GitHub 儲存庫裡的檔案構建網站，並發佈到網路上一個唯一的網址，讓所有人都能查看。

每次當你提交修改到 GitHub 上，Netlify 都會收到通知，並自動重新構建和發布你的網站以應用這些修改。
