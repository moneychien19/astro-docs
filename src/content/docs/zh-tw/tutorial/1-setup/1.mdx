## 獲得你需要的工具

### 終端機

你會需要用到終端機來建立 Astro 專案及使用命令來開發和測試你的網站。

你可以透過作業系統中自帶的終端機來使用命令，常見的工具有 **Terminal** (MacOS/Linux)、**命令提示字元** (Windows) 和 **Termux** (Android)，你的裝置上應該會有其中一個。

### Node.js

為了讓 Astro 可以在你的裝置上運行，你還會需要安裝版本號為 `v18.14.1` 以後的 [**Node.js**](https://nodejs.org/en/)。

你可以在終端機上使用以下命令來確保你已經有相容版本的 `Node.js`：

```sh
node -v

// 輸出範例
v18.14.1
```

如果這個命令返回比 `v18.14.1` 更高的版本號，那麼你就可以繼續了！

如果這個命令返回錯誤訊息（例如 `Command 'node' not found`），或者比 `v18.14.1` 更低的版本號，那你需要先[安裝相容的 Node.js 版本](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)。

### 編輯器

另外，你還需要下載並安裝一個**編輯器**。

:::tip[我們將用到...]
這個教學將會使用 **VS Code**，但你也可以使用適合你作業系統的任何編輯器。
:::

1. [下載並安裝 VS Code](https://code.visualstudio.com/#alt-downloads) 或任何你想使用的編輯器。
