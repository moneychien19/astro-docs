在這個章節，你會將 GitHub 儲存庫連接到 Netlify。在你每次提交修改時，Netlify 將會自動構建並部署此專案到網路上。

:::tip[我們將使用...]
這個教學將會使用 **Netlify**，但你也可以使用你喜歡的網站託管服務來部署網站。
:::

## 新增一個 Netlify 網站

1. 如果你還沒有帳號，請在 [Netlify](https://netlify.com) 創建一個免費帳號。

    記錄下你的使用者名稱，你可以在 `https://app.netlify.com/teams/username` 查看儀表板和你創建的網站。

2. 點擊 <kbd>Add new site</kbd> > <kbd>Import an existing project</kbd>。

    系統會要求你連接到一個 Git 服務，選擇 GitHub 並按照螢幕上的說明驗證你的 GitHub 帳號，然後在列表中選擇你的 Astro 專案的 GitHub 儲存庫。

3. 在最後一步，Netlify 會展示你的網站設定，對於你的 Astro 專案，使用預設值就可以了，所以你可以往下滑動並點擊 <kbd>Deploy site</kbd>。

恭喜你擁有一個 Astro 網站了！

## 修改你的專案名稱

在 Netlify 中，你會在你的網站首頁看到一個隨機生成的專案名稱，以及格式為 `https://project-name-123456.netlify.app` 的網址。你可以將你的專案名稱設為更容易記住的名稱，而這個操作將會自動更新你的網址。

## 前往你的新網站

在你的網站設定中點擊網址，或將其輸入到瀏覽器的網址列來查看你的新網站。
